# SAP 개발 파트너 · 이용 안내 (초안)

**버전:** 2026-05 (검토용)  
**운영:** 캐치랩(Catchy Lab)

본 문서는 현재 배포 기준 기능을 설명하는 초안입니다. 메뉴명·한도·요금은 구독 플랜·관리자 공지에 따라 달라질 수 있습니다.


## 1. 서비스 개요

SAP 개발 파트너는 SAP ABAP **신규 개발**, **분석·개선**, **연동 개발** 요청을 한곳에서 접수하고, AI 에이전트가 **인터뷰 → 개발 제안서(Proposal)** 를 돕는 웹 허브입니다. 필요 시 컨설턴트 회원의 **오퍼·매칭**, **기능명세(FS)·납품 코드**, **최종 구현 산출물** 단계까지 이어집니다.

- **무료·저비용:** 요청 작성, AI 인터뷰, 제안서 생성 등(플랜·AI 크레딧 정책 적용)
- **유료·한도:** FS, 개발 코드, 재생성, 오퍼 등 — 구독 플랜·AI 크레딧 소진
- **언어:** 상단 **한국어 / English** 토글로 화면 문구 전환(일부 관리자 입력은 KO/EN 별도 저장)


## 2. 시작하기

### 2.1 회원가입·로그인

1. **회원가입:** 이메일, 비밀번호, 이름 등 필수 정보 입력 후 이용약관·개인정보처리방침 동의
2. **이메일 인증:** 안내 메일의 링크로 인증(운영 설정에 따름)
3. **휴대폰 인증:** 프로필·알림·체험 플랜 등에 필요할 수 있음
4. **로그인:** 상단 메뉴에서 로그인 · 비밀번호 찾기

### 2.2 Experience(체험) 플랜

- 신규 가입 시 **Experience 체험 기간**이 부여될 수 있습니다(이메일·휴대폰당 1회 등 정책 적용).
- 체험 중 일부 유료 기능 한도가 Junior 등과 동일하게 열릴 수 있습니다.

### 2.3 AI 크레딧

- AI 인터뷰·제안서·FS·코드·AI 문의 등은 **AI 크레딧(원화 잔액)** 을 소진할 수 있습니다.
- **AI 잔액·사용량** 메뉴에서 잔액 확인, **계좌이체 충전 신청**, 신청 취소, 사용·충전 이력을 볼 수 있습니다.
- 입금 확인 후 잔액에 반영됩니다(관리자 확인 절차).


## 3. 홈 화면

| 영역 | 설명 |
|------|------|
| **히어로(좌)** | 서비스 제목·서브카피·설명(관리자 설정) |
| **사용 안내(우)** | 관리자가 입력한 **안내 텍스트**(한 글자씩 타이핑) → 없으면 **YouTube 동영상** → 둘 다 없으면 **ABAP 코드 데모** |
| **상세 보기** | 이용 안내 PDF(기본 `/static/docs/user-guide.pdf` 또는 관리자 지정 URL) |
| **타일 1행** | 공지 · FAQ · 문의/리뷰 — 각 **목록 페이지**로 이동 |
| **타일 2행** | 신규 개발 · 분석·개선 · 연동 개발 — 서비스 소개·신규 요청 |
| **로그인 시** | 타일에 전체·납품·제안·분석·진행중·임시저장 등 **건수** 표시(내 요청 기준) |

관리자는 홈 하단 **「홈 타일 문구 편집」** 또는 **관리자 → 사이트 설정**에서 안내 텍스트·동영상 URL·타일 문구를 수정할 수 있습니다.


## 4. 신규 개발 (RFP)

### 4.1 요청 작성

1. **신규 개발** 타일 또는 메뉴 **RFP** → 작성 마법사
2. **SAP 모듈**(최대 3)·**개발 유형**(최대 3) 선택
3. **요구사항** 입력(평문·리치 텍스트·스크린샷 첨부 가능)
4. **첨부 파일**·**참고 ABAP 코드**(프로그램·섹션 단위) 업로드
5. **임시저장** 또는 **제출**

### 4.2 AI 인터뷰

- 제출 후 **AI 인터뷰** 단계로 이동(최대 3라운드, 라운드당 질문 수 제한).
- 질문에 답변하면 다음 라운드 또는 **제안서 생성**으로 진행.
- **코드 갤러리**와 유사 예제가 매칭되면 질문 품질에 반영될 수 있습니다.

### 4.3 개발 제안서(Proposal)

- 인터뷰 완료 후 AI가 **Development Proposal** 초안 생성.
- 허브 **제안서** 탭에서 조회·인쇄(PDF)·필요 시 **재생성**(크레딧·플랜 한도).

### 4.4 요청 허브(Hub) 단계

통합 허브 URL에서 단계별 탭(진행 상황에 따라 활성화):

| 단계 | 내용 |
|------|------|
| **요청** | 제출한 RFP 본문·첨부·참고 코드 |
| **인터뷰** | 질문·답변 기록 |
| **제안서** | Proposal 본문 |
| **FS** | 기능명세(유료·관리자/정책에 따른 생성) |
| **개발 코드** | 납품 ABAP 등 |
| **최종 구현 산출물** | ZIP 등 단일 파일 업로드·다운로드(권장: ZIP). 이후 요청의 **참고 첨부**로 재사용 가능 |

### 4.5 오퍼·매칭·문의

- **제안서** 단계 등에서 **오퍼 현황** 패널 표시(컨설턴트 오퍼·매칭 상태).
- 요청 회원: 오퍼 **매칭** 선택·취소(납품 존재 시 취소 제한 가능).
- 컨설턴트: **오퍼 제출·철회**, **문의/답변**(이메일·SMS 수신 동의는 오퍼 행에 표시, 컨설턴트 이메일은 노출하지 않음).
- **AI 문의** 플로팅: 요청 맥락 기반 후속 질문(우하단, 크레딧 소진).

### 4.6 대시보드

- **대시보드**에서 내 RFP 목록·필터·정렬·타일/목록 보기.
- 단계 버튼(요청·인터뷰·제안서·FS·코드·산출물)으로 허브 바로가기.


## 5. ABAP 분석·개선

1. **분석·개선** 타일 → 요청 작성(요구사항·ABAP 코드·첨부).
2. 제출 후 **분석 결과** 조회 — 프로그램별·섹션(서브프로그램) 탭으로 코드 구조 확인.
3. 허브에서 **요청 · 분석 · 제안서** 등 단계 이동(워크플로·상태에 따름).
4. 인터뷰·제안서·오퍼·AI 문의·최종 산출물 등은 신규 개발과 유사한 패턴으로 제공될 수 있습니다.


## 6. 연동 개발

1. **연동 개발** 타일 → VBA·Python·API·배치 등 **비-ABAP 연동** 요청 작성.
2. 참고 ABAP·첨부 업로드 시 **프로그램·섹션** 구조로 표시.
3. 인터뷰 → 제안서 → FS·코드·산출물·오퍼 등 **신규 개발과 동일한 허브 단계** 구조.


## 7. 코드 갤러리

- 메뉴 **코드 갤러리**: 관리자가 등록한 ABAP 예제 검색·열람.
- 인터뷰·요청 작성 시 참고 코드로 활용 가능.


## 8. 공지 · FAQ · 문의/리뷰

- **공지 목록/상세**, **FAQ 목록/상세** — 제목·본문 마크다운 지원(다국어 필드 있을 수 있음).
- **문의/리뷰:** 로그인 회원 작성·댓글·별점. 비공개 설정·관리자 비노출 정책 적용.
- 홈 탭에서는 일부만 미리보기, **전체 목록**은 전용 페이지 이용.


## 9. 컨설턴트 회원

### 9.1 가입·승인

- 회원가입 시 **컨설턴트** 유형 선택 → 프로필 첨부 등 추가 제출.
- 승인 전까지 일부 메뉴 제한될 수 있음.

### 9.2 요청 콘솔

- **요청 콘솔:** 본인이 생성하지 않은 **다른 회원의 개발 요청** 목록(정책·매칭 범위에 따름).
- 상세·오퍼·문의 진행.

### 9.3 구독 플랜(컨설턴트)

- Experience · Junior · Senior · Superior 등 — 플랜별 **오퍼·FS·코드** 등 한도 상이(구독 플랜 페이지·관리자 설정 참고).


## 10. 일반 회원 구독 플랜

- Experience · End User · Power User · Process Innovator 등.
- **문의/리뷰**, **개발 요청**, **제안서 재생성**, **FS·코드** 등 기능별 체크 표로 한도 안내.
- 플랜 변경·만료는 운영 정책(계좌이체·관리자 부여 등)에 따름.


## 11. 프로필·계정

- **프로필:** 이름·회사·시간대·비밀번호·이메일 변경(인증)·휴대폰·알림 동의(업무/마케팅, 이메일/SMS).
- **컨설턴트 전환:** 프로필에서 신청(첨부 재업로드).
- **탈퇴:** 유예 기간 후 삭제; 기간 중 이메일로 **탈퇴 취소** 가능.


## 12. 화면·접근성 팁

- **다크/라이트 테마** 전환(상단).
- **처리 중** 오버레이: 폼 제출·긴 작업 시 표시.
- **임시저장** 플로팅 버튼: 작성 화면에서 우측( AI 문의 버튼과 스택 배치).
- 요청 상세·허브에서는 **AI 크레딧 충전** FAB가 숨겨져 AI 문의와 겹치지 않도록 할 수 있음.

- **공지/FAQ/리뷰 모니터링**, **구독 플랜**, **입금·AI 충전 확인**, **코드 갤러리·지식(KB)** 등 — 일반 회원 메뉴와 분리.
