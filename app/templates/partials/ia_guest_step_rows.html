1. 개발 개요

프로그램명: ZSD_MASSOCRE

개발 목적 및 배경

영업팀은 고객 PO를 접수한 뒤 SAP에 판매오더(SO)를 건별로 수기 입력하고 있습니다. 물량이 몰리는 시기에는 하루 반나절 이상 소요되고, 수기 입력 오류로 재작업이 발생합니다.

본 프로그램은 한 화면에서 다수의 주문 데이터를 입력·검증한 뒤 SAP 판매오더를 자동 생성하여, 처리 시간을 단축하고 데이터 정확도를 높이는 것을 목표로 합니다.

기대 효과 (비즈니스 관점)

  • 업무 효율 대폭 향상 — SO 대량 생성 자동화로 수기 대비 처리 시간 최대 80% 단축
  • 데이터 정확도 제고 — SAP 표준 검증 로직 적용으로 입력 오류·재작업 최소화
  • 영업 생산성 향상 — 입력 업무 감소로 고객 대응·전략 업무에 집중

1. Development overview

Program: ZSD_MASSOCRE

Purpose & background

The sales team manually enters customer POs into SAP sales orders one by one. Peak periods take half a day or more and manual errors drive rework.

This program validates multiple order lines on one screen and auto-creates SAP sales orders to cut processing time and improve data accuracy.

Expected business impact

  • Major efficiency gain — up to 80% faster than manual SO entry
  • Higher data accuracy — SAP validation reduces errors and rework
  • Sales productivity — less data entry, more customer and strategy work

1. 목적·범위·전제

1.1. 목적

본 명세는 표준·자체 개발 SAP 테이블·프로그램의 소스코드 및 정의서를 일괄 다운로드하는 ZSDSRCTAB_DL 프로그램 구현 요건을 정의합니다. 개발·운영 업무에 필요한 기술 정보를 일관된 형식으로 신속히 확보하는 것이 목적입니다.

1.2. 범위

본 프로그램은 다음 기능을 포함합니다.

  • 정보 유형 선택 — 프로그램/테이블 유형 선택 및 유형별 입력·다운로드
  • 프로그램 정보 다운로드 — 프로그램 ID 조회, 사용 테이블 자동 식별·표시
  • 소스·정의 산출 — 메인·INCLUDE·텍스트 요소 ZIP, 선택 테이블 정의 Excel

1. Purpose, scope & assumptions

1.1. Purpose

This spec defines ZSDSRCTAB_DL, which downloads source and definition documents for standard and custom SAP tables and programs in a consistent format.

1.2. Scope

  • Type selection — program vs. table workflows
  • Program download — ID lookup, used tables with checkboxes
  • Deliverables — ZIP of sources/texts, Excel table definitions
