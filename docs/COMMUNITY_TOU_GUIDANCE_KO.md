# 커뮤니티(문의/리뷰) 약관·정책 반영 가이드 (초안 작성 시 참고)

법률 자문이 아니라 **서비스 운영·약관 문구 작성 시 포함하면 좋은 항목**을 정리한 내부 메모입니다. 정식 약관은 법무 검토를 권장합니다.

## 반드시 다루면 좋은 주제

1. **운영자의 조치 권한**  
   - 부적절하거나 법령·약관에 위배되는 게시물에 대해 **비공개 처리, 삭제, 계정 제재** 등을 할 수 있음을 명시합니다.  
   - 본 서비스의 **관리자 비노출(`admin_suppressed`)** 기능은 “회원이 공개로 올린 글이라도 운영상 필요 시 타 회원에게 보이지 않게 할 수 있음”에 해당합니다.

2. **게시물의 책임**  
   - 게시 내용에 대한 **민·형사상 책임은 작성자**에게 있음을 밝힙니다.

3. **금지 콘텐츠(예시)**  
   - 타인 비방, 허위 사실 유포, 개인정보 무단 공개, 불법 정보, 스팸·광고 남용, 저작권 침해 등.

4. **신고·이의 제기**  
   - 신고 접수 창구(이메일 등)와 처리 원칙을 간단히 적습니다.

5. **개인정보**  
   - 게시판에 **연락처·계정 정보를 올리지 말 것**을 권고하고, 운영자가 필요 시 게시물을 비공개·삭제할 수 있음을 정책과 연결합니다.

6. **비공개 글의 성격**  
   - 회원이 **비공개**로 올린 글은 **본인과 운영자만** 열람할 수 있음을 명시하면, “1:1 문의 성격”과 맞습니다.

이 파일은 **약관 초안을 AI·담당자가 작성할 때** 위 항목을 빠뜨리지 않도록 하기 위해 저장해 두었습니다.
