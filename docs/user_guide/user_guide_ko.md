# SAP 개발 파트너 · 이용 안내

**버전:** 2026-05-29  
**운영:** 캐치랩(Catchy Lab)

본 문서는 현재 배포 기준 기능을 설명합니다. 메뉴명·AI 단가·수수료는 화면 안내·관리자 공지·이용약관에 따릅니다.


## 1. 서비스 개요

SAP 개발 파트너는 SAP ABAP **신규 개발(RFP)**, **분석·개선**, **연동 개발** 요청을 한곳에서 접수하고, AI가 **인터뷰 → 개발 제안서(Proposal)** 생성을 돕는 웹 허브입니다. 컨설턴트 회원의 **오퍼·매칭**, **기능명세(FS)·납품 코드**, **최종 구현 산출물**, **납품 대금 정산**까지 이어질 수 있습니다.

- **AI 크레딧:** AI 인터뷰·제안서·FS·코드·AI 문의 등은 **선불 AI 크레딧** 잔액에서 차감됩니다(잔액 없으면 해당 기능 제한).
- **납품 대금:** 매칭된 요청에 한해 의뢰자·컨설턴트 간 개발 대금 합의·입금·지급 확인( AI 크레딧과 별도).
- **언어:** 상단 **한국어 / English** 토글.


## 2. 시작하기

### 2.1 회원가입·로그인

1. **회원가입:** 이메일·비밀번호·이름, 이용약관·개인정보처리방침 동의
2. **회원 유형:** **일반 회원**(개발 요청) 또는 **컨설턴트 회원** 신청(승인 전 일부 메뉴 제한)
3. **이메일·휴대폰 인증:** 프로필·알림·충전 등에 필요할 수 있음
4. **로그인** · 비밀번호 찾기

### 2.2 AI 크레딧

- 메뉴 **AI 잔액·사용량**(또는 유사 명칭)에서 잔액·사용·충전 이력 확인
- **계좌이체 충전:** 신청 → 회사 입금 확인 후 반영
- **온라인 결제:** PortOne 등 결제대행(카드·PayPal 등, 화면에 표시된 통화·수단)
- 가입·이벤트 시 무상 크레딧이 부여될 수 있음(화면 안내)


## 3. 홈 화면

| 영역 | 설명 |
|------|------|
| **히어로(좌)** | 서비스 제목·설명(관리자 설정) |
| **사용 안내(우)** | 안내 텍스트 타이핑 → 없으면 YouTube → 없으면 ABAP 데모 |
| **상세 보기** | 본 이용 안내(`/user-guide`) 또는 PDF 링크 |
| **타일 1행** | 공지 · FAQ · 문의/리뷰 |
| **타일 2행** | 신규 개발 · 분석·개선 · 연동 개발 |
| **로그인 시** | 내 요청 건수(전체·납품·제안·진행중·임시저장 등) |

관리자는 **홈 타일 문구 편집** 또는 **관리자 → 사이트 설정**에서 문구·동영상 URL을 수정합니다.


## 4. 신규 개발 (RFP)

### 4.1 요청 작성

1. **신규 개발** 타일 또는 **RFP** 메뉴
2. SAP 모듈(최대 3)·개발 유형(최대 3) 선택
3. 요구사항(평문·리치 텍스트·스크린샷)·첨부·참고 ABAP
4. **임시저장** 또는 **제출**

### 4.2 AI 인터뷰·제안서

- 제출 후 **AI 인터뷰**(라운드·질문 수 제한) → **개발 제안서** 생성
- 허브 **제안서** 탭에서 조회·인쇄(PDF)·**재생성**(AI 크레딧 소진)

### 4.3 요청 허브 단계

| 단계 | 내용 |
|------|------|
| **요청** | 본문·첨부·참고 코드 |
| **인터뷰** | Q&A |
| **제안서** | Proposal |
| **FS** | 기능명세(AI·정책·크레딧) |
| **개발 코드** | 납품 ABAP 등 |
| **최종 구현 산출물** | ZIP 등 업로드·다운로드 |
| **납품·대금** | 매칭 후 대금 합의·결제·지급 확인 |

### 4.4 오퍼·매칭·문의

- 컨설턴트 **오퍼** · 요청자 **매칭** 선택(납품 존재 시 취소 제한 가능)
- **문의/답변**(발신 표시는 정책에 따름, 컨설턴트 이메일 비노출)
- **AI 문의** 플로팅(우하단, 크레딧 사용)

### 4.5 대시보드

내 RFP 목록·필터·허브 단계 바로가기.


## 5. ABAP 분석·개선 · 6. 연동 개발

분석·개선 또는 연동 타일에서 요청 작성 후, 허브에서 분석·인터뷰·제안서·FS·코드·산출물·오퍼·**납품 대금** 단계를 신규 개발과 유사하게 이용합니다.


## 7. 지식갤러리(KB)

- 메뉴 **지식갤러리**(`/kb`): 검수·발행된 **공개 기술 글** 목록·상세
- 회원 요청 원문·납품 소스 전체는 KB에 노출되지 않습니다(요약·일반화된 글만 공개)


## 8. 공지 · FAQ · 문의/리뷰

- 공지·FAQ 목록/상세(한·영 필드 있을 수 있음)
- 문의/리뷰: 로그인 회원 작성·댓글·별점


## 9. 컨설턴트 회원

### 9.1 승인·요청 콘솔

- 가입 시 컨설턴트 신청·프로필 첨부 → **승인 후** 요청 콘솔에서 타 회원 요청 열람·오퍼
- 매칭 건에서 FS·납품·**납품 대금 정산(수령)** 수행

### 9.2 AI 크레딧·납품 대금

- AI 기능 사용 시 본인 **AI 크레딧** 차감
- **납품 대금 정산:** 매칭된 요청 허브의 **납품·대금** 단계에서 합의 금액·플랫폼 결제·지급 확인


## 10. 일반 회원 · 납품 대금(의뢰)

- 매칭 후 허브 **납품·대금**에서 총액 합의·입금(계좌이체 안내 또는 온라인 결제)·지급 확인
- 회사는 회원 간 용역 계약 당사자가 아닙니다(약관 제8조)
