SAP Dev Hub – SaaS Platform (MVP)AI 기반 SAP 개발 자동화 플랫폼의 프론트엔드 웹 애플리케이션입니다.실행 방법```bashcd SAP_SaaS_Platformpip install -r requirements.txtpython -m uvicorn app.main:app --reload --port 8000```브라우저에서 http://localhost:8000 으로 접속프로젝트 구조```SAP_SaaS_Platform/├── app/│   ├── main.py              # FastAPI 앱 진입점│   ├── database.py          # SQLite DB 설정│   ├── models.py            # User, RFP 데이터 모델│   ├── auth.py              # JWT 인증 로직│   ├── routers/│   │   ├── auth_router.py   # 로그인/회원가입 라우터│   │   └── rfp_router.py    # RFP 제출/대시보드 라우터│   ├── templates/           # Jinja2 HTML 템플릿│   └── static/│       ├── css/style.css    # 전체 스타일│       └── js/│           ├── i18n.js      # 한/영 다국어 처리│           ├── main.js      # 공통 JS│           └── rfp_form.js  # RFP 폼 인터랙션├── uploads/                 # 첨부 파일 저장소└── requirements.txt```주요 기능한국어 / 영어 언어 전환 (로컬스토리지에 저장)회원가입 / 로그인 (JWT 쿠키 기반)RFP 요청 수집 폼SAP 모듈 다중 선택 (SD, MM, FI, CO, PP, QM, PM, HCM, WM, PS, EWM, Basis)ABAP 개발 유형 다중 선택 (Report, Dialog, FM, BAdI, BAPI, Upload, Interface, Form, Workflow, Fiori)파일 첨부 (PDF, Excel, Word, 이미지, TXT / 최대 20MB)자유 기술 텍스트사용자 대시보드 (요청 현황 조회)요청 제출 완료 페이지
